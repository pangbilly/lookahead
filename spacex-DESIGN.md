## Overview

Bold, space-age minimalism with dramatic contrast and technical precision. The design feels both futuristic and industrial, emphasizing humanity's reach into space through stark visual hierarchy and clean typography.

Full-width layouts with generous whitespace, centered content alignment, and dramatic scale relationships. The design uses the full viewport for impact while maintaining clean margins and consistent spacing.

Uses a 4px base grid with scale: 0.75, 1.25, 2, 2.5, 3.75, 4.5, 5, 7.5, 10, 15.

## Colors
- **Space Black** (#000000): Main page background, creating dramatic contrast
- **Starlight White** (#f0f0fa): Primary text color for headings and body content
- **Text Secondary** (#ffffff): Secondary text elements and navigation
- **Mission Blue** (#0000ee): Interactive elements and links
- **Border Subtle** (#808080): Subtle borders and dividers

## Typography
- **Headline Font**: D-DIN-Bold
- **Body Font**: D-DIN
- **Label Font**: D-DIN

The typography system uses D-DIN as the primary family, with bold weights for headlines and regular weights for body text. Large-scale headlines feature generous letter-spacing for impact, while body text maintains normal spacing for readability. The weight hierarchy is simple but effective, using size and weight to create clear information architecture.

## Elevation

The design uses no shadows, relying instead on stark color contrast and scale to create hierarchy. Depth is achieved through layered imagery and dramatic scale differences in typography.

## Components
- **Hero Section**: Full-viewport hero with large typography overlay on space imagery, featuring dramatic planetary visuals
- **Navigation Bar**: Horizontal navigation with SpaceX logo and uppercase menu items, transparent background
- **CTA Button**: Minimal outlined button with uppercase text and subtle border styling
- **Launch Counter**: Technical countdown display with monospace-style formatting for mission timing

## Do's and Don'ts
- Do maintain high contrast between text and background for space-tech aesthetic
- Don't use shadows or gradients - keep the design flat and minimal
- Do use generous letter-spacing on headlines for dramatic impact
- Don't mix font families - stick to the D-DIN system
- Do leverage full-width layouts for maximum visual impact
- Don't overcrowd the interface - embrace whitespace as a design element
